# Personal Portfolio Website (Version 1)

##Live Website
https://aaravpersonalsite.vercel.app/

## Overview

This is the first version of my personal portfolio website.
The goal of this project is to create a **minimal, self-built portfolio** that will gradually evolve as I learn new technologies and build more projects.

Rather than using heavy frameworks initially, this version is intentionally built using **vanilla web technologies** to understand the fundamentals of how the web works.

This website will grow over time into a **personal technical playground** containing:

* Projects
* Experiments
* Technical insights
* Learning notes
* Visualizations

---

# Current Version: V1

This version focuses on building a **simple but expressive homepage** that introduces who I am and provides access to my profiles.

### Features

* Custom homepage layout
* Sanskrit quote reflecting personal philosophy
* Profile photo section
* Social links (GitHub, LinkedIn, Instagram)
* Reusable navigation bar component
* Background design with visual identity
* Basic responsive layout for desktop browsers
* Modular folder structure for future expansion

---

# Technologies Used

This project is intentionally built with core web technologies:

* HTML
* CSS
* JavaScript (Vanilla)

No frameworks or UI libraries are used in Version 1.

# Key Design Ideas

The homepage combines:

* A **symbolic background design**
* A **Sanskrit quote representing personal philosophy**
* A minimal **hero section introducing the developer**

The goal is to create a site that feels **personal and evolving**, rather than a static template portfolio.

---

# Current Limitations

This version is intentionally minimal and has some limitations:

* Optimized primarily for **desktop browsers**
* Mobile responsiveness is not fully implemented yet
* Navigation links are placeholders for future sections
* Only the homepage is currently implemented
* Styling and layout will be refined in future versions
